# Quadratic Expression Calculator

Модуль для консольных приложений на C#, вычисляющий значение квадратичного выражения x² + x·y − y² для двух введенных чисел.

## Функциональность

- Ввод двух чисел (могут быть дробными)
- Вычисление выражения: x² + x·y − y²
- Округление результата до двух знаков
- Вывод результата

## Пример работы

Введите два любых числа
5
3
Квадратичное вычитание значений Х и У = 31.00

Введите два любых числа
2.5
1.5
Квадратичное вычитание значений Х и У = 7.75

## Куда можно встроить

- В математические калькуляторы
- В образовательные программы по алгебре
- В системы обработки данных
- В инженерные расчёты

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Вычисление выполняется по формуле: x² + x·y − y². Поддерживаются целые и дробные числа.
